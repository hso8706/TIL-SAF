### 프로젝트명
- tCha
### 기간
- 23.07.10 - 23.08.18
### 기술 스택
- OpenVidu
- java, Spring boot
- Spring Security, Spring Data JPA, AWS RDS(Maria DB), redis, docker
### 서비스 개요
- 웹 RTC 기술을 활용한 온라인 화상 PT 서비스
### 서비스 대상 고객
- 다양한 제약으로 인해 온라인 PT를 희망하는 사용자
- 상세한 PT 기록을 남기길 원하는 사용자
- 커리어를 쌓기 원하는 트레이너 지망생
- 프리랜서로 활동하길 원하는 트레이너
### 서비스 상세 기능
- 웹 RTC를 이용한 화상 PT 서비스 기능
- 운동 중 자세 교정을 위한 드로잉 기능
- 트레이너 즐겨찾기 및 리뷰 기능
### 담당 업무 및 기여한 부분
- Redis를 이용한 정렬 기능 구현
: 정렬 기능의 성능 향상을 위해 Redis 사용
- S3를 이용한 이미지 저장 구현
: 대용량 파일 저장을 위해 AWS S3 사용
- nginx, Let's Ecrypt를 이용하여 https 도메인 설정
: ws, was 분리하여 효율적으로 관리
: https의 사용과 접근성을 향상을 위한 도메인 구매 및 설정
- EC2와 Docker를 이용한 배포
- AWS RDS 설정
### 성과
: 웹 RTC 기술을 활용한 프로젝트 경험
: 기획부터 배포까지 처음으로 진행한 프로젝트 경험
