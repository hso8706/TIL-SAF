# 사용자 입력
### input 키워드
- `input()` 키워드를 사용하여 사용자 입력값을 변수에 할당할 수 있다.
- `input()` 인자로 문자열을 제공할 수 있다. 해당 문자열은 사용자에게 어떤 값을 입력할 수 있는지 안내하는 역할을 한다.
- `input()` 으로 입력받는 값은 기본적으로 `String` 취급을 한다.

### input 여러개 받기
: 기본적으로 받는 데이터 갯수에 해당하는 변수에 할당해야함.
- .split() 사용하기
- map 추가 사용하기
- list 추가 사용하기

# 사용자 출력
### print 키워드
- `print()` 함수 인자 내부에서 연산은 가능하다.
- `print()` 함수 인자는 `콤마(,)`로 구분하고 출력 시 띄어쓰기로 표현할 수 있다.
- `end`, `sep` 키워드를 기억할 것.
