# sys
## stdin
### 사용법
- `from sys import stdin`
//예시. input() => stdin.readline()
### 주의
: sys.stdin.readline()을 사용하면 개행 문자(\n)까지 읽음
: 형변환을 하면 개행 문자를 자동으로 받지 않음(예를 들면, int)
: input().rstrip()을 사용해서 없애는 방법도 있음.

## stdout
### 사용법
- `from sys import stdout`
//예시. print => sys.stdout.write()
