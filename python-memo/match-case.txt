# match-case문
### switch-case문
: java의 switch-case문과 비슷하다. 조금 기능이 추가된 형태이다.  
: python 3.10 이상부터 사용할 수 있는 문법...  
### 기본 형식
```python
match `조건문` :
    case `조건문 결과` :
        `case body`
    ...
    case _ : #`_`가 default 를 의미함
        `case body(default)`
```

# switch-case문
### 파이썬의 switch-case문
: 없다. `if-elif`를 권장하더라.
: 파이썬 3.10 미만이면 조건문으로 사용하자.
