# 스타 트레이너를 만나다

### 유저 입장
- 운동 모르던 시절
  - 헬스장 기웃거리기 씬
  - 이상한 자세로 운동하는 씬
- tcha와의 만남
  - tcha 이용하는 씬
  - 자신감이 넘치는 씬
  - 꾸준함을 보여주는 씬

### 트레이너 계기
- 실제 헬스장
  - 개인 운동 시작
  - 이전의 나와 같은 운동 초보 조우 및 도움
  - 트레이너 진로 흥미
  - 트레이너 공고 확인 후 벽 느낌
- tcha를 회상
  - 아하!(tcha의 문구)
  - 트레이너 등록 후 본인 프로필

### 스타 트레이너 비결
- 비결은 유저 입장을 아는 것.
  - 본인의 경험을 기준으로한 PT 제공
  - 꼼꼼한 피드백
  - 운동일지 잘 써주기
  - 빈 시간대 활용

### 인터뷰 종료
- 후보1. 뉴스 형식의 인터뷰 종료
- 후보2. 도망가기(헬창)
- 후보3. 화면 종료, 스태프와의 대화로 끝
