# 라이브 시연 흐름도(총 7분)

### 1
- 로그인
- 트레이너 등록
- 트레이너 페이지
- 방 생성
- 캘린더

### 2
- 비로그인 상태의 메인 페이지
    - 비로그인 상태 기능(정렬) 하나 소모
    - 검색
    - 예약하기
    - 로그인 유도
- 로그인 후 예약하던 페이지
- 사이드바로 로그인 확인
- 예약 완료
- 트레이너 즐겨찾기 (+ 마이페이지에서 확인 가능)
- 캘린더

### 3
- 라이브 방 둘 다 입장
- 트레이너 종료 (유저 먼저 라이브 퇴장하고, 트레이너 퇴장)
- 라이브 종료 후 운동 일지, 리뷰 쓰기 가능 상태인 거 보여주기
- 유저-리뷰쓰기, 트레이너-일지쓰기
- 트레이너 변경 사항 확인 (낮은 리뷰로 낮아진 점수, 레디스 하나씩으로 변경, 시연 후 변경할 예정)
- 유저 일지 확인
