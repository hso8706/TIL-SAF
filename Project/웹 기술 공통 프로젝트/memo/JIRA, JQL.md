## Jira 사용법

### Issue Type
- Task : 제일 일반적인, 개발과 관련없는 기타 작업
- Story : 실제로 사용자가 영향을 받는 작업 
- Bug : 버그
- Epic : 큰 일, 작업을 묶은 상위 개념

### 필드
- summary : 이슈 제목
- component : 조직, 소속 개념
- desription : 이슈 설명
- repoter : 이슈 제기자
- linked issue : 관련 이슈 연결
- fix version : 고친 작업을 어떤 버전에 설정할 것인지
- assignee : 해당 이슈 담당자
