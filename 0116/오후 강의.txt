# 오후 강의

### 컨테이너
: 여러 개의 값(데이터)을 담을 수 있는 것(객체)  
: 서로 다른 자료형을 저장할 수 있음(미띤; 괴상하군)  
: 컨테이너는 순서가 있는 데이터와 순서가 없는 데이터로 종류를 구분할 수 있다.  
// 순서가 있는 점과 정렬되었다는 말과는 다른 의미인거 알쥐~?  
- 시퀀스형
: 한 줄로 표 가능한 컨테이너(lieaner)  
: 순서가 있는 데이터의 컨테이너  
- 비시퀀스형
: 여러 줄로 표현해야 하는 컨테이너  
: 순서가 없는 데이터의 컨테이너  
- 가변형, 불변형
- 종류
1. 리스트(시퀀스, 가변)
2. 튜플(시퀀스, 불변)
: 거의 잘 안씀
3. 레인지(시퀀스, 불변)
4. 세트(비시퀀스, 가변)
5. 딕셔너리(비시퀀스, 가변)
// 인덱스와 딕셔너리를 주로 쓰게 될 예정

### list
: 한 줄로 정리되어 있음 =>슬라이싱 가능(인덱스로 접근 가능하면 슬라이싱 가능)

### dictionary
- dictionary의 생성
: dic() 혹은 {}

- dictionary의 특징
: 메모리 공간에 여기저기 흩뿌려져 있음.
: 파이썬 3.7 부터는 저장되는 순서를 기억하는 ordered dic이 기본값이 됨.

### set
: 있니, 없니의 유무 정도를 파악
