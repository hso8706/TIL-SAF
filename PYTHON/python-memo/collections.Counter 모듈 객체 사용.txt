# collections.Counter 모듈.객체의 사용법
### collections 모듈  
: 시퀀스형 자료형들에 대한 확장 기능을 제공하는 모듈

### Counter 객체  
: 데이터의 갯수를 파악하는데 특화된 기능을 제공하는 객체  
: Counter 객체의 생성자로 제공하는 시퀀스형 자료형을 분해하여 요소별로 몇 개가 있는지 dictionary 형태로 반환  
: 각각의 요소가 key, 해당 요소가 등장하는 갯수가 value인 dictionary이다.  
: 반환값이 dict이기 때문에 dict의 기능을 모두 적용할 수 있다.

### Counter 생성의 예시
```python

>>> Counter(["hi", "hey", "hi", "hi", "hello", "hey"])
Counter({'hi': 3, 'hey': 2, 'hello': 1})

>>> Counter(["hi", "hey", "hi", "hi", "hello", "hey"])
Counter({'hi': 3, 'hey': 2, 'hello': 1})
```

### .most_common() 메서드  
: Counter 객체의 메서드로써, 객체내의 value에 대하여 내림차순으로 정렬하고 (key, value)를 tuple형식으로 쌍을 짓고 이를 list로 반환한다.
