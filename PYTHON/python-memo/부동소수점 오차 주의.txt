# 부동소수점으로 인한 오차

### 소수와 `==`
- 계산값과 소수의 값이 같다는 조건식을 사용할 땐 주의하자!  
: 계산값과 소수의 값이 같다는 조건식을 위해서 `==`은 사용하면 에러가 날 확률이 매우 높음  
: 예를 들어 `0.1 + 0.2 == 0.3`은 파이썬에선 False가 나온다.

### 해결 방법 키워드
- math.fabs()와 sys,float_info.epsilon
- math.isclose()
- Decimal.decimal()
- Fraction.fractions()
- abs()와 매우 작은 수

### 연계
- `round 반올림 에러, 5사 5입`과 같이 볼 것
