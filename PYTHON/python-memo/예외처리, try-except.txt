- 모든 예외를 다루는 방법
```python
try:
    실행할 코드
except:
    예외가 발생했을 때 처리하는 코드
```

- 특정 예외만 다루는 방법
```python
try:
    실행할 코드
except 예외이름:
    예외가 발생했을 때 처리하는 코드
```

- 예외 에러 코드 받아오기
```python
try:
    실행할 코드
except 예외이름 as 변수:
    예외가 발생했을 때 처리하는 코드
# `변수`를 except 몸체 내부에서 사용할 수 있고, 해당 변수에 except에서 잡힌 예외 에러 코드가 담겨져 있다.
```
