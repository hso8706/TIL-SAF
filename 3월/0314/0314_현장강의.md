### DOM
- Document Object Model
- html 문서를 구성하나를 태그 하나하나를 객체로 취급하여 사용하는 모델
- 객체 : 데이터와 메서드로 구성

### window 지정하기
- window.document : 현재 창의 html 문서
- opener.document : 현재 창을 열어준 html 문서

### 자식 노드와 속성 노드
- 자식 노드는 해당 노드의 하위 계층으로 들어감
- 속성 노드는 해당 노드에 해당됨. 같은 계층 느낌
