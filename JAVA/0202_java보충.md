- 알고리즘용 자바 언어
  - 기본 연산, 기본 문법만 활용
- 프로그래밍용 자바 언어
  - 객체지향 개념 활용

### 배열
- 배열 종류
  - 1차원
  - 다차원

### 환경 설정
- jdk : zulu(jdk 설치 경로를 찾아보면 다양한 부품 확인 가능)
- ide : eclipse

### 경로
- workspace(JavaSup) > project > package

### 선언, 생성, 사용
- 선언 : 정의, 크기 x
- 생성 : 크기 지정
- 사용 : 사용

### 1차원 배열의 구조 종류
```java
int[] a;
a = new int[3];

int[] b = new int[3];
int[] c = {1,2,3};
int d[] = new int[3];
int e; // 불가능, 정의와 동시에 생성
e = {1,2,3};
```

### 2차원 배열
- 2차원 배열 구조
- 2차원 배열 탐색하는 방법 알아두기(4방 탐색, 8방 탐색)
  - 특정 위치를 기준으로 동서남북 + 대각선까지 포함해서 탐색하는 문제 많이 나옴
  - 상(-1,0), 하(+1,0), 좌(0,-1), 우(0,+1)
  ```java
  //예시
  delta 배열: x, y좌표의 변화량을 담은 배열
  int[] dx = {-1, 1, 0, 0};
  int[] dy = {0, 0, -1, 1};
  //dx, dy의 인덱스를 한 쌍으로 사용
  ```
