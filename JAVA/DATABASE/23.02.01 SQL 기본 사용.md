- create database + {DB_name};
  - {DB_name} database 생성 키워드
- use {DB_name};
  - {DB_name} 사용 키워드
- select {option}
  - {option} : * 혹은 {column_name}
  - 조회한 DB_Table 내에서 {option}을 호출하는 명령어  
     ```sql
     # {column_name}이 여러개인 경우 콤마로 구분한다.  
     # 전체를 호출하더라도 *보다 {column_name}을 다 써주는게 빠르다고 한다.
     ```
